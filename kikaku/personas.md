# ペルソナ定義

## ペルソナ1: 山田 義男（62歳）― 年配の現場作業員

| 項目 | 内容 |
|------|------|
| 職種 | 建設現場の作業員（班長） |
| 勤続年数 | 35年 |
| 使用デバイス | Android スマートフォン（息子に勧められて購入） |
| ITリテラシー | 低い。LINEと電話は使えるが、アプリのインストールは家族に頼む |
| 身体的特徴 | 老眼が進行中。小さい文字やアイコンが見えづらい。作業用手袋をしたまま操作することもある |
| 勤怠に関する行動 | 朝は現場に直行するため、到着したらすぐに打刻したい。複雑な操作は避けたい |
| 主な課題 | 画面の文字が小さいと読めない。ボタンが小さいと押し間違える。手順が多いと途中で諦める |
| 求めること | 「開いたら押すだけ」のシンプルさ。大きなボタンと大きな文字 |

## ペルソナ2: 鈴木 恵子（45歳）― 忙しい中間管理職

| 項目 | 内容 |
|------|------|
| 職種 | 営業部 課長。部下15名のマネジメントを兼務 |
| 勤続年数 | 20年 |
| 使用デバイス | iPhone（常に最新モデル） |
| ITリテラシー | 中〜高。業務ツール（Slack, Teams, Excel）は日常的に使用 |
| 身体的特徴 | 特になし。ただし朝の通勤中や移動中にスマホ操作することが多い |
| 勤怠に関する行動 | 出勤・退勤の打刻を最短時間で済ませたい。自分と部下の勤務状況を把握する必要がある |
| 主な課題 | 時間がない。1画面で必要な情報が見えないとストレスを感じる。無駄なステップが嫌い |
| 求めること | 打刻は2秒で完了。勤務時間や残業がひと目でわかる情報密度。効率的なUI |

## ペルソナ3: 佐藤 翔太（23歳）― 新入社員

| 項目 | 内容 |
|------|------|
| 職種 | 情報システム部 新人エンジニア |
| 勤続年数 | 入社1年目 |
| 使用デバイス | iPhone と iPad を併用。個人では Android も所有 |
| ITリテラシー | 高い。大学で情報工学を専攻。日常的に様々なアプリやWebサービスを利用 |
| 身体的特徴 | 特になし |
| 勤怠に関する行動 | 月末に勤務実績を振り返りたい。残業時間を自分で管理し、ワークライフバランスを意識している |
| 主な課題 | 情報が少ないUIだと物足りない。過去の記録をすぐ確認したい。モダンなUIを好む |
| 求めること | データの可視化。月次の勤務実績の一覧。洗練されたデザインとインタラクション |
